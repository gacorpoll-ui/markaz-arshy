## 🖱️ 3 VARIASI CTA

### CTA 1 — Urgency + Curiosity
```

```
> **Logika:** Membuat rasa takut ketinggalan (FOMO) + penasaran dengan penawaran spesifik

### CTA 2 — Personalized Recommendation
```
✅ Upgrade Sekarang — Rekomendasi Spesial untuk Arshy
```
> **Logika:** Personalisasi membuat user merasa diperhatikan, CTA langsung ke aksi

### CTA 3 — Value-First / No Commitment
```

```
> **Logika:** Zero-risk CTA — tidak langsung jual, tapi undang interaksi. Cocok untuk user dengan total belanja masih rendah (Rp 277)
